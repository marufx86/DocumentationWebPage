# Images Folder

Put all your tutorial images and screenshots here.

## How to Use

1. Save your image files directly in this folder
2. Use common formats: `.png`, `.jpg`, `.jpeg`, `.gif`
3. Name files clearly: `pixel-streaming-setup.png` instead of `img001.png`

## Example

If you save a file as `setup-screen.png` here, reference it in your documentation as:

```
media: {
  image: "/images/setup-screen.png"
}
```

## Tips

- Use descriptive names
- Keep file sizes reasonable (under 2MB per image)
- Use PNG for screenshots (better quality)
- Use JPG for photos
